## Installation

### Clone the project
- `git clone git@github.com:fatymahmed/Calculator.git`

### Install dependencies
- `npm install`

### Run the app
 - `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
